Guida Catasto IT GML Merger Pro

Flusso di lavoro:

  1. Scegli la Regione
  2. Seleziona una o più province
  3. Seleziona Tipo di File (MAP/PLE)
  4. Definisci la cartella di destinazione
  5. Definisci i nomi dei file di output
  6. Premi "Elabora"

Suggerimento: Per selezionare più province, usa CTRL+clic.

OSSERVAZIONE: Il plugin lavora in Background, quindi è possibile continuare a lavorare con QGIS!

Tipi di file:

  • Mappe (MAP): fogli catastali
  • Particelle (PLE): particelle catastali

Funzionalità principali:

  • Download automatico dei dati catastali
  • Estrazione e fusione dei file GML
  • Conversione in GeoPackage
  • Caricamento automatico in QGIS
  • Pulizia di attributi inutili
  • Aggiunta degli attributi foglio e particella

Nota: I dati scaricati saranno salvati nella cartella di lavoro.

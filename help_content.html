Guida CatastoIT GML Merger Pro

Flusso di lavoro:

  1. Seleziona il tipo di file da elaborare
  2. Scegli la cartella di lavoro
  3. Seleziona la regione
  4. Seleziona una o più province
  5. Definisci i file di output
  6. Premi "Elabora"

Suggerimento: Per selezionare più province, usa CTRL+clic.

Tipi di file:

  • Mappe (MAP): fogli catastali
  • Particelle (PLE): particelle catastali

Funzionalità principali:

  • Download automatico dei dati catastali
  • Estrazione e fusione dei file GML
  • Conversione in GeoPackage
  • Caricamento automatico in QGIS

Nota: I dati scaricati saranno salvati nella cartella di lavoro.

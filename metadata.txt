about= <b>Catasto IT GML Merger Pro</b> is an advanced QGIS plugin for downloading, extracting and merging Italian cadastral GML files (source: RNDT/AdE — Agenzia delle Entrate). Filter by region, province and municipality via an interactive transfer widget. Outputs GPKG with calculated fields: sheet number (foglio), parcel number (particella), census section (sezione censuaria) and municipality name (comune). Automatically applies a rule-based style to the PLE layer (parcels, roads, water bodies). Output granularity: single file, per province or per municipality. All processing runs in background without blocking QGIS.
	<br><br>
	<b>Catasto IT GML Merger Pro</b> è un plugin avanzato per QGIS che consente di scaricare, estrarre e unire file GML del catasto italiano (fonte: RNDT/AdE — Agenzia delle Entrate). Permette di filtrare per regione, provincia e singoli comuni tramite widget di selezione interattivo. Converte i dati in formato GPKG aggiungendo i campi calcolati: foglio, particella, sezione censuaria e nome del comune. Applica automaticamente uno stile a regole al layer PLE (particelle, strade, acque). Granularità di output: file unico, per provincia o per comune. L'elaborazione avviene in background senza bloccare QGIS.
